Project components

Server-side

Server was written in python with Flask microframework.

Configuration

Server reads configuration values from config.py file, which can be edited manually. Syntax is very easy (usual python syntax), just assign your value to parameter:

UPLOAD_FOLDER - path to folder, which will store your uploaded images.

HOST - hostname of server.

ALLOWED_EXTENSIONS - set of file extensions, which are allowed to be uploaded.

Client

Client was written in C++ with Qt5 framework.

Configuration

Configuration file - config.json

url - URL of server.

screens_folder - path to folder, which will store your screenshots on your localhost.

Manual

Client

How to use? Just call program. In Linux:

$ ./client-qt5

In Windows:

> client-qt5.exe

If you want to take screenshot of part of your screen, you should call client with '-a' command line parameter.

The most convenient way is to create shortcut. Gnome 3: All Settings -> Keyboard -> Shortcuts. Then add 2 new custom shortcuts: one for screenshot of whole screen and other for screenshot of part of screen.

After successful program execution direct link to uploaded image will be in your clipboard. Also, in linux you can receive pop-up notification (requires notify-send).

Curl

It's also possible to upload any image (.png, .jpg, .jpeg or .gif) via curl:

$ curl -F "image=@PATH_TO_IMAGE" HOSTNAME/upload

If successful you'll receive direct link to uploaded image. Example:

http://HOSTNAME/uploaded/f8d7ee72.png
